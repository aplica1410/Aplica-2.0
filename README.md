APLICA — AI-Powered Job Application Email Generation System

APLICA is an AI-powered web application designed to automate professional job application outreach using Artificial Intelligence and Natural Language Processing.

The platform generates personalized and context-aware job application emails dynamically from a provided job description and recruiter details, helping students, freelancers, and job seekers reduce repetitive manual effort during outreach workflows.

AI-powered personalized email generation

Installation & Setup
Clone Repository
git clone https://github.com/aplica1410/Aplica-2.0.git
Backend Setup
Navigate to backend
cd backend
Install dependencies
npm install
Create .env file
OPENAI_API_KEY=your_openai_api_key
PORT=5000
Start backend server
npm start
Frontend Setup
Navigate to frontend
cd frontend
Install dependencies
npm install
Start frontend
npm run dev


**Workflow**
User enters:
Signs Up
Job Description
Frontend sends request to backend
Backend processes request
AI prompt is generated
OpenAI API generates personalized email
Response is returned to frontend
User receives professional outreach email
